# Shibastagram



### **柴犬だけ**を投稿できるSNS
  


## 概要  
柴犬好きのための画像投稿アプリです。画像を選択し投稿ボタンを押すと、柴犬を学習した*AWS　Rekognition*が走り、柴犬かどうかを判定します。現在は、正面かつ一匹のみの写真のみ通ります。柴犬なら投稿が成功し、タイムラインへ反映されます。

## 使用したフレームワーク・言語・技術　
xampp v3.3.0  
laravel8  
PHP8  
python3.8  
AWS S3  
AWS Rekognition  
AWS Lambda  

## 今後実装したい機能　　
相互フォロー限定のDM機能　　

フォローしたユーザーの投稿をタイムラインのトップに表示　　

投稿にユーザーアイコンを表示　　

柴犬と判定される範囲を広げる(横顔、二匹以上など)
